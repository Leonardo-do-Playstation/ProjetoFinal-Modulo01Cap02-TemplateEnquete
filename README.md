🗳️ Template de Enquete

Projeto web que apresenta um template de enquete (poll) desenvolvido com HTML e CSS, permitindo estruturar perguntas e opções de votação em uma interface simples e organizada.

O projeto foi criado com o objetivo de praticar estruturação de páginas web, criação de formulários e estilização de elementos interativos utilizando CSS.

🚀 Tecnologias Utilizadas

HTML5

CSS3

📌 Funcionalidades

✔ Estrutura de enquete com perguntas e opções
✔ Layout organizado para votação
✔ Estilização com CSS
✔ Interface simples e fácil de usar

⚙️ Como Executar o Projeto

1️⃣ Clone o repositório

git clone https://github.com/Leonardo-do-Playstation/ProjetoFinal-Modulo01Cap02-TemplateEnquete.git

2️⃣ Acesse a pasta do projeto

cd ProjetoFinal-Modulo01Cap02-TemplateEnquete

3️⃣ Abra o arquivo index.html no navegador

Ou utilize uma extensão como Live Server no VS Code.
